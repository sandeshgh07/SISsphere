# Security Policy

## Supported Versions

Document which versions currently receive security updates.

## Reporting a Vulnerability

Please do not report security vulnerabilities in public issues.

Please report vulnerabilities directly to: **sandeshgh07@gmail.com**

Include:
- A clear description of the issue.
- Steps to reproduce.
- A proof of concept if available.
- Impact assessment.

## Response Process

- We will acknowledge receipt as soon as practical.
- We will investigate and validate the report.
- We may ask for more detail.
- We will coordinate disclosure after a fix is available.
